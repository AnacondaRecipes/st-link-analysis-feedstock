Installing st-link-analysis
==================

Installing `st-link-analysis` from the main channel can be achieved by:

```
conda install st-link-analysis
```

It is possible to list all of the versions of `st-link-analysis` available on your platform with `conda`:

```
conda search st-link-analysis
```
